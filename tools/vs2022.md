# 개발 환경
- **IDE**: Visual Studio 2022 {{에디션}}
- **VS 경로**: `C:/Program Files/Microsoft Visual Studio/2022/{{에디션}}/`
- **MSBuild**: `C:/Program Files/Microsoft Visual Studio/2022/{{에디션}}/MSBuild/Current/Bin/MSBuild.exe`
- **MSBuild 스위치 접두사**: MSBuild 호출 시 스위치 접두사를 `/` 대신 `-`로 사용할 것. bash 셸에서 `/`가 경로로 해석될 수 있음.

## 에디션별 기본 경로
- Community: `C:/Program Files/Microsoft Visual Studio/2022/Community/`
- Professional: `C:/Program Files/Microsoft Visual Studio/2022/Professional/`
- Enterprise: `C:/Program Files/Microsoft Visual Studio/2022/Enterprise/`

## 경로 탐지
1. 위 기본 경로에서 에디션 폴더 존재 여부를 Enterprise → Professional → Community 순으로 확인
2. 기본 경로에서 찾지 못한 경우 vswhere로 탐지:
   ```
   "C:/Program Files (x86)/Microsoft Visual Studio/Installer/vswhere.exe" -version [17.0,18.0) -property installationPath
   ```
